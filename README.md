# DevTasks ✅

A simple beginner-friendly task tracker built with React + Tailwind CSS.

## 🚀 Features
- Add tasks
- Mark as completed
- Delete tasks

## 🛠️ Tech Stack
- React
- Tailwind CSS

## 📦 Installation
```bash
git clone https://github.com/your-username/devtasks.git
cd devtasks
npm install
npm run dev
```
